Box-shadow

O que é?

Utilizamos o box-shadow para dar uma sombraao objeto selecionado: div, span, images, qualquer elemento HTML

Deslocamento da sombra

é importante que a base da sombra (x e y) estará sempre positiva, então podemos definir aonde será apresentada

Exemplo:

Sombra no lado direito: box-shadow: 20px 0;

Sombra no lado esquerdo: box-shadow: -20px 0;

Sombra abaixo do objeto: box-shadow: 0 20px;

Sombra acima do objeto: box-shadow: 0 -20px;

A compreensão e mistura dos objetos acima faz ela variar para outras direções

img { box-shadow: 10px 20px; }

Desfoque:

Para desfocar um objeto incluímos um última valor ao box-shadow

Quanto maior o valor do desfoque, mas a sombra ficará desfocada

img { box-shadow: 10px 20px 30px; }

Expansão:

Por padrão, a sombra tem um mesmo tamanho do elemento

É possível EXPANDIR a sombra para todas as direções. Adicionando mais um valor ao box-shadow

A expansão aumenta a sombra para todos os lados, sendo possível alterar as pontas

OBS: O uso do valor de expansão é OPCIONAL, mas se for utilizado o DESFOQUE deve ter o valor zerado. (Boa prática)

img { box-shadow: 10px 20px 0 40px; }

Cor:

É possível alterar a cor da sombra, que tem como padrão a cor preta. É mais um valor inserido no box-shadow

img { box-shadow: 10px 20px 30px 40px red; } img { box-shadow: 10px 20px 30px 40px #ff0000; } img { box-shadow: 10px 20px 30px 40px RGB(255,0,0); }

INSET (Sombra interna)

É comum termos a sombra sempre externa ao objeto, mas podemos ter sombra interna

O valor inset funciona com divs, spans e outros elementos, MENOS com imagens

div { box-shadow: 10px 20px 30px 40px #ff0000 inset; } div { box-shadow: inset 10px 20px 30px 40px #ff0000; }
