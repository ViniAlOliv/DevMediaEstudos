BOX SIZING

Numa situação na qual tenho uma imagem que, se eu incluir um espaçamento interno, vá ultrapassar a página

O box-sizing ajuda a limitar a borda de acordo com o tamanho máximo da página, sem criar barras de scroll

O que é?

Propriedade do CSS que altera a forma como o tamanho do elemento é calculado

Exemplo: Temos uma padding de 20px e largura de 350px. O total é de 390px (350 + 20 + 20)

Se temos uma largura de 350px, o box-sizing fará o ajuste para: x + 20 + 20 = 350

Logo x é 310, já que se considera o padding junto com o tamanho total da largura.

Uso

O valor do box-sizing é o border-box. Temos outros, que não são usados.

O uso de box-sizing pode ser visto como uma maneira simples de garantir que os tamanhos definidos no width/height sejam respeitados.

div { width: 250px; padding: 30px; box-sizing: border-box; }
