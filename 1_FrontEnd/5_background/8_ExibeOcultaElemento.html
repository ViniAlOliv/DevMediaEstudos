Exibindo e ocultando elementos

Block e Inline

Todo elemento HTML possui ou o tipo Block, ou o tipo Inline

Block

Bloco, apresentando um acima do outro. Títulos, textos e divs são exemplos

Inline

Em linha, apresentado ao lado do outro. Span, link e a imagem

DISPLAY

Podemos mudar um elemento para linha ou bloco, com o Display. Que usará 'block' ou 'inline'

O display none remove o espaço do elemento da tela!

div { display: inline; } span { display: block; } img { display: none; }

VISIBILITY

Tem duas propriedades muito usadas: visible e hidden

div { visibility: visible; } span { visibility: hidden; }

OPACITY

Deixa o objeto opaco, trabalhando com sua transparência

div { opacity: 100%; } span { opacity: 50%; } img { opacity: 0%; }
