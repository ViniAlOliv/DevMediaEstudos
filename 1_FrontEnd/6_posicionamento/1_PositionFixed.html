Position Fixed

Introdução

Por vezes, ao acessar um site, vemos:

  • Uma barra de título fixa, que desce junto com o scroll
  • Um botão que fica ativo, ou até mesmo um acesso ao chat

Tudo é feito com o position fixed

Fixando elementos na prática

Para fixarmos um elemento devemos usar o CSS como no exemplo abaixo:

.div_fixa { position: fixed; border: 2px solid red; }

Movendo elementos

Para movermos um elemento fixo devemos incluir as propriedades:

top: 0px; bottom: 0px; left: 0px; right: 0px;

BOA PRÁTICA: Utilizar px e indicar o position antes

Exemplo:

.div_fixa { position: fixed; border: 2px solid red; right: 100px; }
