Link de CSS pode aser acidionado no Head, acima. A tag HEAD define características da página, como título ou descrição da mesma.

A tag META define a descrição de uma página, como incluído acima. A tag META pode ou não conter o / para seu fechamento.

A tag META Charset especifica o uso de caracteres, como o UTF-8
