# Algoritmo

- Programar
    Realizar uma sequência de passos para construir softwares, como programas, sites, páginas e etc...

- Por que programar:
    Fazer o computador entender a série de comandos, um passo a passo

- Programador:
    Pessoa que atua programando

- Plano de estudo:
    Lógica, Algoritmo e Linguagem de Programação

## O que é algoritmo:
- Série de passos ordenados para alcançar um objetivo
- Primeiro passo de estudos de programação

Erros que devem ser evitados (Boas práticas):
1. Não detalhar um passo -> Como o Sistema entenderá algo?
2. Pular uma etapa -> Software precisa entender passos anteriores para chegar ao fim
3. Ordem incoerente -> A ordem deve ser vista e revista
4. Definir um passo incompleto

- Dicas:
Descubra qual o objetivo (problema).
Comece a se questionar como resolver esse problema.
Responda as perguntas que forem surgindo até não ter mais nenhuma.
Baseado nas perguntas e respostas crie os passos (algoritmo) até chegar no objetivo ou resolver o problema.

## O que é lógica?
Significa adicionar automações nos passos de um algoritmo
O primeiro passo para criar um software é escrever um algoritmo com o que ele deve fazer.
Algoritmo são os passos que você quer que o software faça.
Lógica adiciona automações ao algoritmo. Por exemplo, podemos fazer com que o algoritmo passe a tomar decisões durante a sequência de passos.

A cada passo lógico que inserimos no nosso algoritmo melhor e menos suscetível a erros ele será
