Definindo o Tamanho dos Elementos

Sabemos como definir uma Largura. Agora, vamos definir o tamanho máximo e mínimo da mesma.

  • min-width
  • max-width

img { width: 100%; min-width: 1200px; max-width: 1300px; }

Acima, definimos que a imagem não pode ter menos que 1200px

Também, que não poderá ser maior que 1300px

Sempre que as propriedades min e max forem definidas, sobrepõem as definidas no width

Altura

Para altura, mesma coisa acima. Temos o min-height e max-height

.imagem-noticias { height: auto; min-height: 480px; max-height: 640px; }

Boas práticas

max e min podem ser usados de forma independete, não precisando sempre ter os dois.
