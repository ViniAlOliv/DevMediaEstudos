Valores Relativos

Digamos que tenhamos uma imagem com width de 1440px. Dependendo da tela, aparecerá uma barra de rolagem. Colocando em 100% ele aproveitará o tamanho da tela

É importante termos um elemento pai que nos dê um LIMITE para imagens.

O ponto chave é: Determinar um valor total Limite. Dentro desse total, a porcentagem

Exemplo: Uma div que recebe um valor de 1000px, e eu tenho uma imagem dentro que pega 90%

Para utilizar a porcentagem, basta passar o valor com % na propriedade.

div { width: 800px; }

img { height: 50%; padding: 3% 8%; }

Qualquer propriedade que receba valor em pixel pode receber em %

VW e VH

Esses valores são VALORES RELATIVOS À TELA.

VW -> Value to Width

VH -> Value to Height

Razão de cálculo é igual ao %

Calc

Podemos ter uma foto que ocupe 100% da página. Se adicionarmos uma borda, a página ultrapassará o limite de tela

Ele calcula o valor relativo de algo. Exemplo:

Imagem ocupando 100%, mas incluí borda de 30px. Logo, são 60px de borda

img { width: calc(100vw - 60px); }
