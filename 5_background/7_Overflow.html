Overflow

Imagine que temos um texto muito grande e esse texto ultrapasse o limite (Já vi isso em alguns sites)

o Overflow é para evitar que esse tipo de coisa aconteça

div { overflow: scroll; }

Difereças de perspectivas:

SCROLL: Usado para limitar texto. Cria uma rollbar mesmo quando o conteúdo não é ultrapassado

AUTO: Usado para limitar texto. Só cria a rollbar se o conteúdo passar

Overflow x e y

div { overflow-x: scroll; overflow-y: scroll; }

O overflow-x inclui a barra de rolagem na horizontal

O overflow-y inclui a barra de rolagem na vertical

OBS:

p { overflow: auto scroll; }

p { overflow-x: auto; overflow-y: scroll; }

Ambos códigos fazem a mesma coisa

OBS: Dvemos falar de FlexBox:

p { display: flex; }
